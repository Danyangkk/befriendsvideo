# Befriendsvideo — 短视频带货创作多智能体平台

**[English](./README.md) | 中文**

**在线演示：** https://danyangkk.github.io/befriendsvideo/

一个 AI-Native 的多智能体创作平台：把头部短视频创作专家的方法论翻译成系统架构——输入"商品链接 + 创作想法"，产出"创作要素报告 + 多条含分镜的带货脚本"，让每个普通创作者具备专家级创作能力。

## 架构速览
- **1 个主控（CLAUDE.md）+ 5 个能力 SubAgent + 1 个离线采集 Agent**，基于开源 Hermes Agent 框架
- **会话黑板 + 6 类状态失效契约**：状态驱动路由——"用户改了什么 → 哪些产物失效 → 从最早失效处重算、其余复用"
- **13 个标准化 SKILL**：注册表治理 + 技能候选沉淀回路，系统越用越专家
- **爆款资产数仓**：双库 × 每品类 Top100、可解释爆款分；三层召回（品类硬过滤 → 向量粗筛 → LLM 精排）
- **5 条系统闭环**：创作 / 修改 / 灌库 / 技能沉淀 / 效果验证
- **4 类过程指标先于研发设计**：确认闸一次通过率、微调轮次、审核首次通过率、脚本导出率

## 关于此演示
这是一份**高保真交互原型**，用于交互方案验证与研发对齐。**所有数据均为演示 mock，无后端。**生产前端将以此原型为视觉 spec 重写。

体验路径：首页点击示例填入商品链接 → 走完创作流 → 在确认闸输入"钩子换成提问式"体验局部重算 → 确认生成脚本与分镜 → 右上角切换"超管"角色查看效果指标看板与技能注册表。
